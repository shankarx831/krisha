# Radioform Marketing Site

Next.js marketing and documentation site for Radioform.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript

## Development

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Deployment

Designed for deployment to Vercel or Cloudflare Pages. Build is isolated from macOS-specific components.

```bash
npm run build
```
